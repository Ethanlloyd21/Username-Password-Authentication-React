# Username/ Password Authentication using React

## Dependencies

``` bash
npm install @okta/okta-signin-widget --save

npm install @okta/okta-react react-router-dom --save
```

## Quick Start

``` bash
# Install dependencies
npm install

# Serve on localhost:3000
npm start

# Build for production
npm run build
```
